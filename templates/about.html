⚙️ How It Works

1
User enters any URL in the input box
↓
2
System extracts 87 features from the URL (length, dots, https, subdomains etc)
↓
3
Features are passed to Random Forest ML Model
↓
4
Model compares with patterns learned from 11,430 real URLs
↓
5
Result shown — Safe or Phishing with confidence score

🤖 ML Model Used

This project uses Random Forest Classifier — an ensemble learning method that uses 100 decision trees to make predictions. It was trained on 11,430 URLs and achieved an accuracy of 96.94% on test data.

🛡️ Why This Project?

Phishing attacks are one of the most common cyber threats today. Millions of people lose their personal data and money every year due to fake websites. This tool helps users verify any URL before clicking on it — protecting them from cyber attacks.

📊 Dataset Used

The model was trained on the Web Page Phishing Detection Dataset from Kaggle — containing 11,430 URLs with 87 features each, including both legitimate and phishing websites from real world data.
